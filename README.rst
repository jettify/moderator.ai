Moderator Demo
==============

Install the app::

    $ cd moderator
    $ pip install -e .

Run application::

    $ make run

Open browser::

    http://127.0.0.1:9001


Requirements
============
* aiohttp_
* aiohttp_jinja2_


.. _Python: https://www.python.org
.. _aiohttp: https://github.com/aio-libs/aiohttp
.. _aiohttp_jinja2: https://github.com/aio-libs/aiohttp_jinja2
