Moderator AI Demo
=================

Install the app::

    $ cd moderator
    $ pip -r requirements-dev.txt
    $ pip install -e .

Build UI::

    $ yarn install
    $ make build

Run application::

    $ make run

Open browser::

    http://127.0.0.1:9001


Requirements
============
* aiohttp_


.. _Python: https://www.python.org
.. _aiohttp: https://github.com/aio-libs/aiohttp
